# PLANTAMIZ-2025-
Le jeu que nous vous proposons cette année comme projet informatique est inspiré du jeu
"FarmHeroes" que vous trouvez gratuitement sur smartphone. Ceux qui auront la curiosité de
tester ce jeu comprendront le nom de ce projet. Adapté pour convenir au mode console, le jeu
garde malgré tout l'esprit du jeu d'origine dans son principe : aligner des items pour les faire
disparaître (on retrouve également ce principe dans le célèbre CandyCrush).
